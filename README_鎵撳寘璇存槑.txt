客户通讯录导入助手 - Android Studio 打包说明

一、软件功能
1. 导入 .xlsx Excel 文件：默认读取第一个工作表，A列为姓名/Full Name，B列为手机号/mobile Num，从第2行开始读取。
2. 数据先存入App内部数据库，不会立即写入手机通讯录。
3. 支持新增、编辑、删除、搜索、筛选联系人。
4. 支持创建分组，导入 Excel 时选择分组；每个分组内联系人从序号 1 开始排列。
5. 支持按 5人、10人、20人、50人、自定义人数批量写入手机系统通讯录。
6. 支持按具体分组的序号区间写入手机通讯录。
7. 写入成功后，状态自动更新为“已导入”；下次不会重复导入。
8. 支持号码异常、重复号码识别。
9. 支持导出当前列表为 CSV，Excel 可直接打开。

二、打包APK步骤
1. 在电脑安装 Android Studio。
2. 解压本压缩包。
3. 打开 Android Studio，点击 File -> Open，选择 CustomerContactImporter 文件夹。
4. 第一次打开需要等待 Gradle Sync。若提示需要安装 SDK 35 或 Gradle 组件，点击 Install/Download/Sync 即可。
5. 同步完成后，点击顶部菜单：Build -> Build Bundle(s) / APK(s) -> Build APK(s)。
6. 打包成功后，APK 一般在：
   app/build/outputs/apk/debug/app-debug.apk
7. 把 app-debug.apk 发送到小米/安卓手机，允许安装未知来源应用后安装。
8. 第一次批量导入通讯录时，手机会弹出通讯录权限申请，请选择“允许”。

三、使用方式
1. 打开App，点击“导入Excel”选择 .xlsx 文件；或在微信中打开Excel后选择“用其他应用打开/分享”到本App。
2. 选择导入到哪个分组，可选择已有分组或新建分组。
3. 确认导入结果：新增待导入、号码异常、重复号码。
4. 选择顶部具体分组后，可按顺序导入 5/10/20/50 人、自定义人数，或按序号区间导入。
5. 在联系人列表中可编辑、删除、单独导入。
6. 通过顶部搜索框、分组筛选框和状态筛选框查看未导入、已导入、异常号码、重复号码。

四、当前默认规则
1. A列内容作为手机通讯录联系人姓名。
2. B列内容作为手机号码。
3. 手机号只保留数字；+86 或 86 开头会自动处理。
4. 11位且以1开头的号码视为有效手机号。
5. 手机号相同视为重复号码。
6. 重置已导入只会修改App内部状态，不会删除手机通讯录中的联系人。
7. 清空数据只会清空App内部数据库，不会删除手机通讯录中的联系人。

五、后续可扩展功能
1. 增加公司名、地区、来源、行业等字段。
2. 增加导入前预览和勾选导入。
3. 增加去手机通讯录中检测是否已存在的功能。
4. 增加真正的 .xlsx 导出。
